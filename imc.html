O Índice de Massa Corporal (IMC) é uma medida que relaciona o peso e a altura de uma pessoa para avaliar se ela está dentro de faixas consideradas saudáveis.É uma ferramenta simples e amplamente utilizada para identificar se uma pessoa está abaixo do peso, com peso normal,com sobrepeso ou obesidade.
